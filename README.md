# 8086 Instruction Set Reference

A modern, responsive documentation reference for the Intel 8086 microprocessor.

This project modernizes how assembly documentation is viewed. Instead of legacy table-based layouts, it uses a structured, bookshelf-style interface that makes browsing technical specifications intuitive across all devices.
