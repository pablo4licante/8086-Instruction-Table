# 8086 Instruction Set Reference

A modern, responsive documentation reference for the Intel 8086 microprocessor.

This project aims to modernize the experience of reading assembly documentation. It replaces legacy, table-based layouts with a structured "shelf" interface, making it easier for students, developers, and hobbyists to browse technical specifications on any device.

üöÄ Key Features

Visual Flag Register: Flags (ZF, CF, etc.) are clearly visualized with color-coded indicators, allowing developers to instantly see which processor states are modified, set, or cleared by an instruction.

Single-File Architecture: The entire application uses a modern frontend approach but compiles into a single HTML file. It requires no installation, no build steps, and works entirely offline.

Search & Navigation: Instructions can be filtered instantly by mnemonic or category, with a collapsible sidebar for efficient browsing.

Dark Mode Interface: A high-contrast, eye-friendly theme inspired by modern IDEs (Treesitter/Neovim) for comfortable reading during long coding sessions.
